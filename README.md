# handball-training-qc

Es un proyecto público con el objetivo de aprender a realizar evaluaciones de LLMs centrado en la capacidad de responder preguntas tácticas de balonmano.

# Objetivo

La idea de este proyecto es poder responder preguntas abiertas y poder validar si las respuestas que nos dan los LLM a estas preguntas abiertas son buenas o no. Al hacerlo de preguntas abiertas, añadimos cierta complejidad a la hora de evaluar porque al ser preguntas que no tienen una respuesta binaria (SÍ/NO) , tenemos que ayudar a entender al modelo qué es lo que entra dentro del rango de respuestas correctas ante este tipo de preguntas
